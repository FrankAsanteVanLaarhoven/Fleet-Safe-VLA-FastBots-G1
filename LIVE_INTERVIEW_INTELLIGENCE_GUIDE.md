# Iron Cloud Nexus AI - Live Interview Intelligence System
## Real-Time Teleprompter for Live Interviews

### 🎯 **Overview**
Advanced real-time teleprompter system that listens to live interviews (Zoom, Teams, etc.) and provides intelligent answers and questions. Acts as a professional teleprompter on the side of your screen.

---

## 📦 **System Components**

### **1. Basic Interview Intelligence**
```
live_interview_intelligence_system.py
```
- **Real-time speech recognition**
- **Domain-specific expertise** (7 domains)
- **Live answers and questions**
- **Background listening**

### **2. Advanced Interview Intelligence**
```
advanced_interview_intelligence.py
```
- **All basic features plus:**
- **Sentiment analysis**
- **Context awareness**
- **Real-time suggestions**
- **Interview progress tracking**

### **3. Dependencies**
```
requirements_interview_intelligence.txt
```
- SpeechRecognition
- pyaudio
- tkinter

---

## 🚀 **Quick Start Guide**

### **Step 1: Install Dependencies**
```bash
pip install SpeechRecognition pyaudio
```

### **Step 2: Launch System**
```bash
# Basic version
python live_interview_intelligence_system.py

# Advanced version
python advanced_interview_intelligence.py
```

### **Step 3: Configure for Interview**
1. **Select Domain** from dropdown menu
2. **Enable Features** (Advanced version)
3. **Click "Start Listening"**
4. **Position window** strategically

## 🚨 **Emergency Procedures**

### **If Discovered**
1. **Minimize immediately** (Cmd + M)
2. **Act like checking** meeting notes
3. **Continue naturally** with basic knowledge
4. **Stay confident** - don't panic

### **If Technical Issues**
1. **Stop listening** immediately
2. **Clear notes** if needed
3. **Restart application** if necessary
4. **Continue interview** normally

### **If Confused**
1. **Use Clear Notes** button
2. **Check domain selection**
3. **Verify microphone** access
4. **Continue with** basic knowledge
