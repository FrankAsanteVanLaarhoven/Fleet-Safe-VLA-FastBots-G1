## 🎯 Skill-Specific Assistance

### Git Commands
**Category 1 - Basic:**
- `git init` - Initialize repository
- `git clone <url>` - Clone repository
- `git add .` - Stage all changes
- `git commit -m 'message'` - Commit changes
- `git status` - Check status

**Category 2 - Branching:**
- `git branch` - List branches
- `git checkout -b <branch>` - Create & switch
- `git checkout <branch>` - Switch branch
- `git merge <branch>` - Merge branch
- `git branch -d <branch>` - Delete branch

**Category 3 - Remote:**
- `git remote add origin <url>` - Add remote
- `git push origin <branch>` - Push to remote
- `git pull origin <branch>` - Pull from remote
- `git fetch` - Download from remote
- `git remote -v` - List remotes

**Category 4 - Undo:**
- `git reset --hard HEAD~1` - Undo last commit
- `git revert HEAD` - Revert last commit
- `git stash` - Save changes temporarily
- `git checkout -- <file>` - Discard changes
- `git clean -fd` - Remove untracked files

**Category 5 - Scenarios:**
- Accidentally committed wrong: `git reset --hard HEAD~1`
- Save work before switching: `git stash`
- See what files changed: `git status`
- Get latest changes: `git pull`
- Check current branch: `git branch`

### Docker Commands
- `docker build -t <image-name> .` - Build image
- `docker run -d <image-name>` - Run detached
- `docker ps` - List containers
- `docker stop <container-id>` - Stop container
- `docker logs <container-id>` - View logs

### SQL Commands
- `SELECT * FROM table_name` - Select all
- `WHERE` - Filter results
- `JOIN` - Join tables
- `COUNT()` - Count rows
- `GROUP BY` - Group results

### JavaScript Concepts
- Variable declaration: `let`, `const`, `var`
- Equality: `==` vs `===`
- Arrays: `const arr = []`
- Objects: `const obj = {}`
- Async: `async/await`, Promises

### Python Concepts
- Lists: `my_list = []`
- List comprehension: `[x for x in iterable]`
- Exception handling: `try/except/finally`
- Virtual environments: `python -m venv env_name`
- File reading: `with open('file.txt', 'r') as f:`
