## 🎯 **Stealth Modes Explained**

### **🔧 System Monitor Mode**
- **Looks like:** Legitimate system monitoring tool
- **Best for:** Docker, JavaScript, Java
- **Usage:** Click "Refresh" to see skill assistance in "process list"
- **Behavior:** Act like monitoring system performance

### **🧮 Calculator Mode**
- **Looks like:** Real scientific calculator
- **Best for:** Git, Mathematical Reasoning
- **Usage:** Do calculations first, then press "=" for assistance
- **Behavior:** Act like doing mathematical work

### **📝 Notes App Mode**
- **Looks like:** Simple notes application
- **Best for:** SQL, Python, Data Science
- **Usage:** Click "Clear" to see skill assistance as "notes"
- **Behavior:** Act like reviewing personal notes
